# CyberSec Roadmap

Free structured cybersecurity learning path for SOC Analysts, Pentesters, Malware Analysts, and Security Engineers.

👉 **https://gnaixnaij.github.io/cybersec-roadmap/**

## Tracks

| Role | Skills | Levels |
|------|--------|--------|
| 🛡️ SOC Analyst | Log analysis, SIEM, IR, threat hunting | Beginner → Intermediate → Advanced |
| 🔓 Penetration Tester | Recon, exploitation, AD, cloud | Beginner → Intermediate → Advanced |
| 🔬 Malware Analyst | Reverse engineering, YARA, deobfuscation | Beginner → Intermediate → Advanced |
| ⚙️ Security Engineer | Detection, cloud security, IaC, compliance | Beginner → Intermediate → Advanced |

## Features

- **Progress tracking** — Skills save to your browser's localStorage
- **Free only** — Every resource is free, no paid courses listed
- **Curated** — Hand-picked resources that actually teach well
- **Connected** — Links to Cyber Cheatsheet, Deobfuscator, and Malware Decode Lab

## Local dev

```bash
git clone git@github.com:gnaixnaij/cybersec-roadmap.git
cd cybersec-roadmap
# Open index.html in a browser — no build step needed
```
